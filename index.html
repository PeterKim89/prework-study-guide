HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Header tags should be made in order from h1, h2, h3... etc.
  • You can tag specific sections with a unique id to identify them later.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The "." modifier designates the following class as a class selector and will apply properties to respective classes in the HTML file.
  • You can control the size of any element by modifying their width, length, margin, border, adding a box-shadow, etc.
  • You can use inline css in certain cases, but a stylesheet is usually preferred.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new brand and switches to it
  • git add -A: stages all new changes made in the working branch
  • git commit -a: commits any files that have been staged with git add and any other file changes that have been made since then
